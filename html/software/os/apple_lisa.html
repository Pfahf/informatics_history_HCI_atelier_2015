APPLE LISA

With the Apple Lisa Applications Development Tool Kit, programmers were able to create Lisa-style programs that extended the system's power even further. The Lisa operating system featured preemptive multitasking and protected memory - extremely advanced features for a personal computer at the time. Partly based on advanced elements from the failed Apple III SOS operating system released three years earlier, the Lisa also organised its files in hierarchal directories, making the use of large hard drives practical. The Macintosh eventually adopted this disk organisational design as well for its HFS filing system. Conceptually, the Lisa resembles the Xerox Alto in the sense that it was envisioned as an office computing system. Consequently, Lisa has two main user modes - the Lisa Office System and the Workshop. The Lisa Office System is the GUI environment for end users. The Workshop was a program development environment and was almost entirely text-based, though it used a GUI text editor. The Lisa Office System was eventually renamed "7/7", in reference to the seven supplied application programs: LisaWrite, LisaCalc, LisaDraw, LisaGraph, LisaProject, LisaList, and LisaTerminal

LisaProject

Lisa project was the first graphical project management software. Developed for the Apple Lisa computer, LisaProject was conceived and implemented by Debra Willrett of SoloSoft and developed for Apple's Lisa computer. The Lisa operating system features cooperative (non-preemptive) multitasking and virtual memory, then extremely advanced features for a personal computer. The use of virtual memory coupled with a fairly slow disk system makes the system performance seem slow at times. The Apple Lisa Office System was released in 1983 by Apple Computers with their Lisa microcomputer. The Lisa hardware, operating system, and its set of office applications were designed by Apple to work together to create a powerful document processing workstation. It was collected because it is representative of a period when the Lisa and the Macintosh competed against each other, with this software marking the terminal period in the life of the Lisa.
